## LINK INSTALL 

##  FOR UBUNTU 24 Keatas :
Masukkan perintah dibawah jika anda menggunakan OS Ubuntu Version 24 Keatas
```
wget --no-check-certificate https://raw.githubusercontent.com/genznet/Genznet-VPN/main/c1.sh && chmod +x c1.sh && ./c1.sh
```
## JIKA ERROR GANTI MENU :

```
wget --no-check-certificate https://raw.githubusercontent.com/genznet/Genznet-VPN/main/c1.sh \
&& (sed -i 's/\r$//' c1.sh || dos2unix c1.sh) \
&& chmod +x c1.sh \
&& ./c1.sh
```

## FOR UBUNTU DI BAWAH 24 
Masukkan perintah dibawah jika anda menggunakan OS Ubuntu Dibawah Version 24
```
apt install -y && apt update -y && apt upgrade -y && apt install lolcat -y && gem install lolcat && wget https://raw.githubusercontent.com/genznet/Genznet-VPN/main/c1.sh && chmod +x c1.sh && ./c1.sh
```
